Graphic Design Challenge Results

#1 Photo with Text

Text overlay onto a photo. The photo is from cell phone and is the corner of a child's painting. Overlay done with Fotor.com.

#2 Photo with Text

Another cell phone photo taken on the second morning of challenge. Text overlay done with Canva.com.

#3 First SVG

An extension of a tutorial. The top left one was the result of the tutorial. Extended to four versions with a bit of color play. Plus getting a copy of the browser screen display of the SVG code into a PNG.

#4 US Flag SVG

Patriotic theme. Rectangles with overlays. And then the stars. Polygons, sizing, and placement.

#5 Bear Face SVG

Inspired by one I saw on another site. All of my own code. Circles are easy but this required learning about placement, overlaying, and the SVG Path element. The bear's muzzle is in two parts.

#6 Cherry Blossom Facebook Cover

My Facebook cover pic for Easter. Done at Canva.com.

#7 Super Pink Moon on Starscape

Wife took photo of the Super Pink Moon. I created the starscape and added the moon to it. Used Gimp for starscape and to crop and add moon.

#8 Jumbotron Graphic for New Blog

Based on yesterday's pink moon graphic, this Jumbotron will be used on my new blog. Added text with Gimp.

#9 PacMan SVG Graphic

Pacman SVG created with Inkscape. Needed a bit of fun today.

#10 Be Counted - Census 2020

Stand up and be counted. #EveryoneCounts. Created with Canva.

#11 Rather Be Coding logo

Rather Be Coding logo for my new blog. Moon is pic my wife took. Text and png created with Inkscape.

#12 Texture Play in InkScape

Texture play with Inkscape. Textured background and text.

#13 Rock faced tile in InkScape

Rock faced block face created with Inkscape.

#14 Simple Cloud

Simple cloud created with Inkscape. Multiple gradients, duplicate, union and difference.

#15 Outlined Text Using Offset

Text with multiple outlines created using offsets with Inkscape.

#16 Simple Arrow with dual gradients

Simple Arrow with dual gradients and outline. Created with Inkscape.
